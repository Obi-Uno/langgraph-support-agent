Policy knowledge base (RAG)

  • Shipping — 3-5 day standard, $15 expedited, no cancellation once shipped
  • Returns & refunds — 30-day window, refunds over $500 need manager sign-off, damaged items need a ticket
  • Account & billing — email/address changes, subscription cancellation rules

Guardrails to try

  • Ask something off-topic (a recipe, coding help) — rejected before the main model runs
  • Use angry or legal wording ("this is a scam", "I'll sue") — escalates to a specialist
  • Ask it to open a ticket — nothing is written until you click Approve

Behind the scenes

Live audit trail — every guardrail decision, tool call and approval is logged

Send a message to watch the agent pipeline work:

• guardrail block — off-topic queries are rejected by a cheap classifier before the main LLM ever runs
• tool call — read-only lookups execute instantly
• awaiting approval — write actions pause the graph until a human decides
• escalation — angry/legal language or big refunds route to a specialist
