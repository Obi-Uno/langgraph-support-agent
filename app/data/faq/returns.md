# Return & Refund Policy

Customers can request a return within 30 days of delivery for a full refund,
provided the item is unused and in its original packaging.

Refunds over $500 require manager sign-off before being processed — the agent
should never confirm a refund over this amount itself, only create a ticket
and explain that a specialist will follow up within 1 business day.

Damaged or defective items are eligible for a replacement or refund regardless
of the 30-day window, but always require a support ticket with photos, so the
agent should create a ticket rather than resolve this itself.
